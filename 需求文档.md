## 快捷命令

我要创建一个可以记录快捷命令的网页，基于下面的诉求进行设计：

要求

- 界面风格类似notion，界面比较美观现代
- 要编写完整的功能使用介绍

要具备以下功能

- 可以将当前内容导出成excel表格
- 可以将excel表格中的内容导入进来
- 文档中的命令方便复制
- 文档中的命令方便检索
- 方便归类整理
- 界面中的命令可以拖动

---

## 项目使用说明

### 启动项目

双击运行 `启动项目.bat`，脚本会自动：
1. 检查并安装依赖（如果 node_modules 不存在）
2. 复制 sql.js 相关文件到 public 目录
3. 启动开发服务器（后台运行）

启动后访问 http://localhost:3000 使用应用。

**注意：启动窗口需要保持打开，关闭窗口将停止服务器。**

### 关闭项目

双击运行 `关闭项目.bat`，脚本会：
1. 终止端口 3000 上的进程
2. 终止所有 node.exe 进程
3. 关闭启动脚本的窗口

### 版本管理

项目使用 Git 进行版本控制，重要版本标记如下：
- **v1.0.0** - 初始版本，基础功能实现

- **v4.0.0** - UI 优化：现代化标题栏、侧边栏交互改进

### 技术栈

- Vue 3 + Vite

- sql.js（本地数据库）
- xlsx（Excel 导入导出）
